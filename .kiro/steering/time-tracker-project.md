# Time Tracker 项目信息

## 项目概述
个人时间管理 APP，用于记录、分类、统计每天的时间使用情况。

## 技术栈
- 平台：Android（手机 + 平板）
- 框架：Flutter
- 语言：Dart

## 核心功能

### 1. 时间记录
- 记录时间段（开始时间、结束时间）
- 给记录命名、分类、打标签

### 2. 常用事件模板
- 预设常用事件（吃饭、睡觉等）
- 一键快速记录

### 3. 统计可视化
- 日/周/月/季度视图
- 趋势图（时间线变化）
- 饼图（分类占比）
- 按标签统计
- 按时间段统计（如每天上午都在干嘛）

### 4. 目标追踪
- 设定目标（如每天学习 2 小时）
- 追踪完成情况

### 5. APP 限制
- 软禁用指定 APP
- 设定特定时间才能打开

### 6. UI 要求
- 好看、现代化的界面
- 手机和平板自适应
